Which short is the best?

Insertion Sort - Usada com apenas alguns itens, pequenos, etc.
Bubble Sort - Provavelmente nunca. Usada para fins educacionais para ensinar sort.
Selection Sort - Provavelmente não usaremos. Usada para fins educacionais.
Merge Sort - Rápido. Se você se preocupa com worst case cenarios, é o melhor. Não é bom se você se preocupa com space complexity.
Quick Sort - Melhor que merge sort, mesma velocidade do merge, e menor space complexity. A desvantagem é o worst case, cuidado ao escolher.
